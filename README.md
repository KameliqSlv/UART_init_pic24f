# UART_init_pic24f

Това е embedded система за мониторинг и контрол на температура, реализирана с:

PIC24FJ256GA702

ADC – четене на температура и setpoint

UART – комуникация с компютър

GPIO – управление на изход (LED / реле)

Системата:

чете температура (AN0)

чете зададена стойност (AN1)

сравнява ги

управлява изход (RA2)

изпраща данни по UART
